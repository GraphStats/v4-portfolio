# New Year Theme

## Active Window

December 20 00:00:00 -> January 2 23:59:59

## Visual Features

- Gold color palette
- Animated text gradient
- Subtle glow accents
- Seasonal decorative overlay

## Technical Implementation

- Runtime detection in `components/special-theme-handler.tsx`
- Theme styles in `styles/special-themes/new-year.css`
- Public CSS copy in `public/styles/special-themes/new-year.css`

## Notes

- Theme activates automatically within the configured window.
- No manual action is required for normal operation.
- Effects are tuned for performance and readability.
